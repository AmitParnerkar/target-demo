Target Demo

"Upload and Parse a Spreadsheet with Products to PostgreSQL" - Swagger API
PostgreSQL running in docker, auto-deployed to any cloud (AWS/GCP/Azure) - GitHub Code

This project showcases
simple API
using apache poi, spring boot & postgreSQL.

Upload a spreadsheet with products and it will be parsed and stored in PostgreSQL.
Swagger API is available to test the API.

Target Demo API

Upload a spreadsheet with products using a POST and then use a GET end-point to retrieve all products as json.

Automated dev-ops

Domain name to deployment in cloud is all done via terraform with infrastructure code. Docker and kubernetes is used.
